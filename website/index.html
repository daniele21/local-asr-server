FAQ

Only the detail you need, when you need it.

Does PrivateMeet upload audio or transcripts?

No. The core recording, transcription, analysis and search workflow is designed to run locally. Any future optional external integration would be clearly identified and disabled by default.

Which Macs are supported?

The first release is designed for Apple Silicon Macs. 16 GB of unified memory is recommended. Final support for 8 GB devices depends on performance testing.

Does it work with Zoom, Meet and Teams?

Yes. PrivateMeet captures microphone and system audio, so the core workflow is not tied to one meeting platform.

Do I need to invite a bot?

No. PrivateMeet runs as a native Mac application and does not enter the meeting as an external participant.

Is recording always legal?

No universal rule applies. Recording laws and consent requirements vary by jurisdiction. The user remains responsible for informing participants and obtaining the required consent.
