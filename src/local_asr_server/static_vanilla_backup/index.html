ROUTING AUTOMATICO

Audio del computer

Il routing audio viene configurato automaticamente alla registrazione.

Quando premi Avvia registrazione, il sistema:

  1. Rileva l'uscita audio corrente
  2. Crea un dispositivo Multi-Output temporaneo
  3. Registra sia le cuffie che BlackHole
  4. Ripristina l'uscita originale al termine

Requisito: BlackHole 2ch installato (brew install blackhole-2ch).
